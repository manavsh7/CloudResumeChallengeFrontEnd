Tax Portal Web App

TaxPortal is a comprehensive solution designed to offer secure access to tax documents and facilitate efficient client data management. Built with integrated AWS services such as S3, Lambda, CodePipeline, and IAM, the platform ensures seamless file handling, robust security, and scalability. The application incorporates role-based access control, allowing administrators to assign time-limited permissions to employees, ensuring that sensitive information remains protected. Detailed audit logging enhances transparency and accountability, enabling users to track file access and modifications effectively.

  • React
  • AWS S3
  • AWS Lambda
  • AWS DynamoDB

DMV Slot Appointment Checker

This Python script leverages Selenium WebDriver to automate the process of scraping the DMV website for available appointment slots. It continuously monitors the site for new openings and compares them to user-defined preferences. When a desired timeslot becomes available, the script sends an email notification, ensuring users are promptly informed and can secure their appointment without constant manual checking.

  • Python
  • Selenium WebDriver

Measurement App

The Door/Window Measurement App is a mobile application designed to streamline order processing and customer management for a local door and window installation business. Built using React Native and AWS services like Amplify, Cognito, and DataStore, the app automates the creation of orders based on input measurements. Employees can also generate Excel files for the recorded orders, providing an efficient way to manage and share order details. This solution enhances operational efficiency by simplifying workflows and ensuring seamless data storage and retrieval.

  • React Native
  • AWS Amplify
  • AWS Cognito
